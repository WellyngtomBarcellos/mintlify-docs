# <Icon icon="gear" iconType="thin" color="#3e20ff" size={40} /> Configurações

Com o sistema Lhama, você tem algumas ferramentas poderosas, incluindo a conversão de formato até a compressão e insersão de legendas.

- Compressão de 0 a 100% baseada na ualidade atual do video.
- Mais de 10 formatos de videos disponiveis para conversão e compactação
- Video pesado ? sem problemas, com o lhama você tem uma máxima compressão sem reduzir a qualidade total.

## <Icon icon="golf-flag-hole" iconType="thin" color="#3e20ff" size={40} /> **Vai que é sua**

API construida com 3 simples intensoes:

1. \*\*Direto: \*\*Cada rota faz exatamente oque você lê:

   ```bash
   post: api/video/compress/global (Comprime com parâmetros globais)
   post: api/video/compress/partial (Comprime com alguns parâmetros parciais)
   ```
2. \*\*Consistente: \*\*Nossa API retorna sempre um option contendo o erro ou os dados de retorno, o que facilita a implementação em linguagens de tipagem forte e constistência em todas as operações.
